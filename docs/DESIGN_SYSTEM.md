# 🎞️ Project: Me Before You Design System

## 1. Design Concept
"The Last Record of Us" - 영화의 한 장면 같은 애틋한 기록과 따뜻한 연결.

## 2. Color Palette
- **Accent (#A52A2A):** 루이자 클라크의 레드 드레스. 강조, 핵심 버튼, 활성화 상태.
- **Background (#FDFDFD):** 깨끗하고 따뜻한 미색. 전체 배경.
- **Primary (#1A1A1A):** 부드러운 블랙. 가독성 높은 메인 텍스트.
- **Paper (#F0EDE4):** 종이 질감 베이지. 카드 배경이나 섹션 구분.
- **Soft (#EEEBDE):** 연한 테두리선.

## 3. Typography Hierarchy
- **Heading:** Serif (Italic 권장) - 영화 타이틀 느낌.
- **Body:** Sans-serif - 가독성 중심.
- **Label:** Uppercase + Tracking-widest - 잡지 캡션 느낌.

## 4. UI Rules
- **Corner Radius:** `3rem` (둥근 모서리 강조)
- **Shadow:** 깊이감 있는 그림자로 입체감 부여.
- **Interaction:** 모든 전환은 Framer Motion을 활용한 부드러운 Fade & Slide.
