### 📱 User 페이지 (`app/auction/page.tsx`)
- **입찰 로직 강화:**
  - 입찰 성공 시 유저 테이블의 `balance`를 즉시 차감하여 실시간 긴장감 조성.
  - 트랜잭션 처리(로그 저장, 아이템 갱신, 잔액 차감) 안정화.
- **UI/UX 구성:**
  - **상단:** 고정 잔액바(Sticky Header)로 내 자산 상시 확인.
  - **중앙:** 거대 가치관 카드 배치로 제목 가독성 극대화.
  - **하단:** 전체 가치관 목록 배치.
  - **안내 모달:** 최초 진입 시 쿠션어를 활용한 우아한 경매 가이드 노출.
